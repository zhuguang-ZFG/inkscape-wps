# Inkscape-WPS AI 提示词指南

本文档为 inkscape-wps 项目提供完整的 AI 提示词集合，帮助 AI 助手更好地理解和处理项目开发、维护、优化等任务。

## 项目背景

**inkscape-wps** 是一个类似 WPS/Word 的桌面端写字机上位机软件，主要用于控制 GRBL(XYZ) 写字机。项目采用核心算法与 UI 分离的架构设计，支持多种单线字库格式。

### 核心特性
- WPS 风格界面（Ribbon + 三组件：文字/表格/演示）；**主界面文案以中文为主**（技术类字符串仍可能为英文）
- 支持多种字库格式（Hershey、JSON、JHF、奎享导出）
- GRBL 协议完整支持（串口通信、流式传输）；**设备** 页可选 **Z 抬落笔** 或 **M3/M5**（`gcode_pen_mode` / `gcode_m3_s_value`）
- 矢量图形导入和位图矢量化
- **工程文件**（`*.inkwps.json`，格式版本2）：支持文字/表格/演示/插入矢量保存加载；**机床配置**分离（`machine_config.toml` / `.json`）；**原子写入**（`project_io` 临时文件 + `replace`），**G-code 导出**复用 **`write_text_atomic`**
- 配置持久化（JSON/TOML，优先TOML）
- 跨平台快捷键支持（macOS Cmd / Windows Ctrl）
- 表格单元格富文本编辑（HTML格式）
- 根目录 **`README.md`**：`python3 -m inkscape_wps`、**SPEC.md** / **AI_PROMPTS.md** 索引、`unittest discover` 命令

### 架构约束
- 核心层 (`inkscape_wps/core/`)：禁止使用 PyQt6/PySide，仅用标准库
- UI层 (`inkscape_wps/ui/`)：可使用 PyQt6，通过桥接模式与核心交互
